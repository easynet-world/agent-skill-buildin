# HTML Report Template (Bootstrap 5)

The company report **must** be a single, well-formed HTML file using **Bootstrap 5** for a professional, polished look. Use the CDN links below and the structure in this document.

## Bootstrap 5 CDN (required in `<head>`)

```html
<link href="https://cdn.jsdelivr.net/npm/bootstrap@5.3.2/dist/css/bootstrap.min.css" rel="stylesheet" />
<script src="https://cdn.jsdelivr.net/npm/bootstrap@5.3.2/dist/js/bootstrap.bundle.min.js" defer></script>
```

## Document structure

```html

<html lang="en">
<head>
  <meta charset="UTF-8" />
  <meta name="viewport" content="width=device-width, initial-scale=1.0" />

<body>
```

## Required elements

- `<!DOCTYPE html>`, `<html lang="en">`, `<head>`, `<body>`.
- **Bootstrap 5 CSS** in `<head>`: `https://cdn.jsdelivr.net/npm/bootstrap@5.3.2/dist/css/bootstrap.min.css`.
- **Bootstrap 5 JS** before `</body>` (optional but recommended): `https://cdn.jsdelivr.net/npm/bootstrap@5.3.2/dist/js/bootstrap.bundle.min.js`.
- Main content wrapped in `<div class="container py-4">` (or `container-fluid` if you prefer full width).
- Exactly one `<section id="company-overview">` with a **card**: `card` > `card-header` ("1. Company Overview") + `card-body`.
- Exactly one `<section id="stock-data">` with a **card** containing a Bootstrap **table**: `table table-striped table-hover` inside `table-responsive`.
- Exactly one `<section id="news">` with a **card** and a **list** of news items (e.g. `list-group` or `list-unstyled` with links).
- Exactly one `<section id="sources">` with a **card** and a list of source links.
- Use Bootstrap utility classes for spacing (`mb-4`, `py-4`), text (`text-muted`, `text-success`, `text-danger`), and typography (`display-5`, `fw-bold`).
- All tags properly closed; escape `&`, `<`, `>` in text.

## Styling tips

- **Header:** Use `display-5` or `display-6` for the report title, `text-muted` for the date.
- **Cards:** Use `card`, `card-header` (e.g. blue background: `bg-primary` or custom `.card-header { background-color: #0d6efd; color: #fff; }`), `card-body`.
- **Stock table:** Use `table table-striped table-hover`; for positive change use `text-success`, for negative use `text-danger`.
- **News:** Each item as `list-group-item` or a `<li>` with an `<a>` and optional `<small class="text-muted">` for date/source.
- **Footer:** Use `text-center text-muted small` and `border-top` for a clean separator.
- Optional inline `<style>` for report-specific tweaks (e.g. header border, card shadow, news item left border).
